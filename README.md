Excel Tool for Map Visualisation - Production of Sorghum in West Africa
(Microsoft Excel workbook with macros)

Description
This repository contains an Excel workbook (West Africa Sorghum Data Mapper.xlsm) developed to:
- filter and visualize sorhum production data on a gridded map,
- visualize distinct production area clusters for selecter scenarios
- export data into csv file for processing
The file includes VBA macros, which must be enabled for full functionality.

Usage
Download the West Africa Sorghum Mapper.xlsm file.
Open it in Microsoft Excel (desktop version recommended).
Enable macros when prompted.
Use the selection cells and colouring buttons on the map sheets (Map / Clustering Map).

Requirements
Microsoft Excel
Macro execution enabled
